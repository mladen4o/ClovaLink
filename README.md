## üìã Overview

ClovaLink is a Rust-powered, multi-tenant file storage system. It features secure APIs, extensible metadata, and support for various storage backends, such as S3, Wasabi, and local volumes. It's tailored for self-hosting, making it an ideal choice for enterprises that prioritize control and flexibility in file management.

### Key Features

- **Multi-Tenant Architecture**: Host multiple users while keeping their data secure.
- **Extensible Metadata**: Customize file metadata to suit your needs.
- **Secure APIs**: Access your files safely through robust APIs.
- **Flexible Storage Options**: Use S3, Wasabi, or local storage based on your preference.
- **Pluggable Backends**: Easily switch or add new storage backends.
